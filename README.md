# Local Loop Experiment — Meta-Canary

This is the working repository for the **Local Loop experiment week**, a structured seven-day sprint to build a verified-node obligation graph for the Merseyside local economy.

The **meta-canary** is the rehearsal session that validates the working method before the main experiment begins. This repository is the single source of truth for all planning documents, prompts, method artefacts, and build outputs produced during the week.

## Repository Structure

| Directory | Purpose |
|---|---|
| `/planning` | Working brief, VSM map, and rolling plan |
| `/method` | Working method protocol and daily state documents |
| `/prompts` | All agent prompts (coaching, session charter, handoff, coding) |
| `/log` | Raw method log: charters, handoffs, signals, observations |
| `/artefacts` | Diagrams, rendered outputs, photos, and scans |
| `/specs` | BDD feature specifications, data model, non-functional requirements |
| `/code` | All source code for the Local Loop build |
| `/tickets` | Backlog, in-progress, and done tickets |
