## Definition of Done — Meta-Canary

The meta-canary is complete when:

1. A working QR code generator is deployed and accessible at `meta-canary.commoner.services`
2. The application is password-protected and accessible only to the MCS/Local Loop team
3. A user can generate a QR code by entering a URL, and the QR code encodes that URL directly
4. Previously generated QR codes are stored persistently and viewable in the application
5. QR codes can be exported as PNG and SVG files
6. The application is free to run (no per-use fees; hosted on a free tier)
7. The working method has been exercised end-to-end: session charters, handoff notes, BDD behaviours, one-page specification brief, coding agent commits with rich messages, end-of-day batch, and an updated rolling plan
